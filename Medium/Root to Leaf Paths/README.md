<h2><a href="https://www.geeksforgeeks.org/problems/root-to-leaf-paths/1">Root to Leaf Paths</a></h2><h3>Difficulty Level : Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 18px;">Given a Binary Tree of size N, you need to find all the possible paths from the root node to all the leaf nodes of the binary tree.</span></p>
<p><span style="font-size: 18px;"><strong>Example 1:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input:</strong>
       1
    /     \
   2       3
<strong>Output: <br></strong>1 2 <br>1 3 
<strong>Explanation: </strong>
All possible paths:
1-&gt;2
1-&gt;3
</span>
</pre>
<p dir="ltr"><strong><span style="font-size: 18px;">Example 2:</span></strong></p>
<pre><span style="font-size: 18px;"><strong>Input:
&nbsp;        </strong>10
&nbsp;      /    \
&nbsp;     20    30
&nbsp;    /  \
&nbsp;   40   60<strong>
Output: <br></strong>10 20 40 <br>10 20 60 <br>10 30 
</span>
</pre>
<p dir="ltr"><span style="font-size: 18px;"><strong>Your Task:</strong><br>Your task is to complete the function&nbsp;<strong>Paths()</strong> which takes the root node as an argument and returns all the possible paths. (All the paths are printed in new lines by the driver's code.)</span></p>
<p><span style="font-size: 18px;"><strong>Expected Time Complexity:&nbsp;</strong>O(N).<br><strong>Expected Auxiliary Space:&nbsp;</strong>O(H).</span><br><span style="font-size: 18px;"><strong>Note:&nbsp;</strong>H is the height of the tree.</span></p>
